📊 E-Commerce Sales Data Analysis
📌 Project Overview

This project analyzes a large E-Commerce sales dataset containing 186,850 records to uncover meaningful insights into sales performance, customer purchasing behavior, and product demand. The goal of this project is to transform raw transactional data into actionable business insights using Python and data analysis techniques.

The dataset contains important information such as Order ID, Product, Quantity Ordered, Price Each, Order Date, and Purchase Address. Through data cleaning, preprocessing, and exploratory data analysis (EDA), this project identifies key sales trends that can help businesses improve decision-making and optimize their strategies.

🎯 Project Objectives

The main objectives of this project are:

Analyze sales performance across different months

Identify the best performing sales period

Determine the most popular products

Explore patterns in customer purchasing behavior

Visualize insights using charts and graphs

🛠 Tools & Technologies Used

Python

Pandas – Data cleaning and manipulation

Matplotlib – Data visualization

Seaborn – Advanced statistical visualization

Jupyter Notebook – Analysis environment

📂 Dataset Information

The dataset includes the following columns:

Column	Description
Order ID	Unique order identifier
Product	Product name
Quantity Ordered	Number of units purchased
Price Each	Price per product
Order Date	Date and time of purchase
Purchase Address	Customer purchase location

Total Records: 186,850

🔍 Data Cleaning Process

Before performing analysis, several preprocessing steps were applied:

Removed missing values

Eliminated duplicate rows

Converted data types for numerical calculations

Extracted month information from order date

Created a sales column (Quantity × Price)

These steps ensured the dataset was accurate and ready for analysis.

📈 Key Insights
📅 Best Month for Sales

The analysis shows that December generated the highest sales revenue, exceeding $4.6 million. This indicates strong seasonal demand, likely due to holiday shopping.

🛍 Most Popular Products

The most frequently purchased products are:

AAA Batteries (4-pack)

AA Batteries (4-pack)

USB-C Charging Cable

Lightning Charging Cable

Wired Headphones

These products have high demand and consistent sales volume.

💡 Business Insight

Small accessories such as cables and batteries appear frequently in orders, suggesting they are essential add-on purchases.
